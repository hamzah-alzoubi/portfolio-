# 💼 Portfolio Website

## 📌 Overview

This is a personal portfolio web application built using **ASP.NET Core MVC**.
The project showcases my skills, projects, and experience in web development through a clean and responsive interface.

---

## 🚀 Features

* 🔐 User authentication system
* 🧑‍💼 Personal profile and about section
* 📂 Projects showcase with details
* 📬 Contact form
* 📱 Responsive design (works on all devices)

---

## 🛠️ Technologies Used

* ASP.NET Core MVC
* C#
* SQL Server
* Entity Framework Core
* HTML5 / CSS3
* Bootstrap

---

## ⚙️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/hamzah-alzoubi/portfolio.git
   ```

2. Open the project in Visual Studio

3. Update the database:

   * Open **Package Manager Console**
   * Run:

     ```bash
     Update-Database
     ```

4. Run the project:

   * Press **F5** or click **Start**
